# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺：
- 对所有参与者保持友好和包容
- 尊重不同的观点和经验
- 建设性批评时保持优雅
- 关注对社区最有利的事情
- 与社区其他成员协作

## 我们的责任

社区维护者有责任：
- 明确解释可接受行为的标准
- 对不可接受的行为采取公平、适当的纠正措施
- 对违反本准则的行为给予适当警告或暂时/永久封禁

## 不可接受的行为

以下行为在社区中不可接受：

- 公开或私下骚扰、歧视
- 人身攻击或贬低性评论
- 恶意挑衅、侮辱/贬低评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他不道德、不专业的行为

## 我们的职责

社区维护者有权利和责任：
- 删除、编辑或拒绝与本准则不符的评论、提交、代码、文档 Issue 等
- 暂时或永久禁止任何违反本准则的参与者
- 在适当情况下，以公正、透明的方式做出决定

## 适用范围

本行为准则适用于：
- 所有官方社区空间（GitHub、邮件列表、Discord 等）
- 代表 AgentNet 社区的个人行为

## 联系方式

如发现违规行为，请通过以下方式联系我们：
- **GitHub Issues**: https://github.com/agentnet/protocol/issues
- **邮件**: conduct@agentnet.ai

所有投诉将经由社区治理团队审查和调查，并给出公平、适当的回应。
